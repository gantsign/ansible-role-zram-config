Ansible Role: Zram Config
=========================

Role to install Zram Config for compressed RAM swap.

Requirements
------------

* Ubuntu

Example Playbook
----------------

```yaml
- hosts: servers
  roles:
     - { role: gantsign.zram-config }
```
